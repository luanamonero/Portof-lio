## Instruções

1. Clone o repositório
  * `git clone git@github.com:luanamonero/portfolio`.
  * Entre na pasta do repositório que você acabou de clonar:
    * `cd portfolio`

2. Instale as dependências
  * `npm install`

3. Inicie o projeto 
  * `npm start`
